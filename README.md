# Aplicação Portifolio Flutter

Aplicação feita em flutter para praticar e desenvolver
um portifólio web.

# Aprendizado proposto

* Layout Responsivo
* MediaQuery
* Personalização de componentes
* Aplicação de patterns como o DRY (Don't Repeat Yourself)
* Export para web e desktop
* Utilização de fontes personalizadas
* Utilização de libs terceiras
